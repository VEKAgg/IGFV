# Squadron Page Images

This directory should contain the following images for the Squadron page:

1. `stars-bg.jpg` - A dark starfield background image
2. `space-nebula.jpg` - A colorful nebula space background
3. `planet-texture.jpg` - A texture for the planet in the Join section
4. `video-poster.jpg` - A poster image for the video section

These images should be high-resolution and optimized for web use. The background images should be dark-themed to match VEKA's UI style with orange accents.

## Image Specifications

- Background images: 1920x1080px minimum, JPG format
- Planet texture: 1024x1024px, JPG format
- Video poster: 1920x1080px, JPG format

Please add these images to enable the full visual experience of the Squadron page.
