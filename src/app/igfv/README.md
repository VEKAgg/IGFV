# IGFV (Interstellar Goodfellas) Page

This page showcases the Interstellar Goodfellas squadron for Elite Dangerous. It features a modern, interactive design with smooth animations and parallax effects.

## Features

- **Animated Header**: 
  - "Interstellar Goodfellas" title with an oscillating ship icon
  - Ship icon continuously moves left and right until the final section
  - Responsive text sizing for all devices

- **Interactive Elements**:
  - Framer Motion animations for smooth transitions
  - Parallax scrolling effects
  - Scroll-triggered animations
  - Hover effects on buttons and cards

- **Sections**:
  1. Hero Section
     - Animated title with ship icon
     - Call-to-action buttons
     - Scroll indicator
  2. About Section
     - Squadron history and description
     - Parallax background effects
  3. Activities Grid
     - Combat Operations
     - Trading & Mining
     - Exploration
  4. Join Section
     - Discord integration
     - Inara.cz link

## Technical Implementation

- Built with Next.js App Router
- TypeScript for type safety
- Tailwind CSS for styling
- Framer Motion for animations
- React Scroll Parallax for parallax effects
- Responsive design for all screen sizes

## Required Assets

Background images in `/public/images/`:
- `stars-bg.jpg` - Dark starfield background (1920x1080px minimum)

To run the development server:
